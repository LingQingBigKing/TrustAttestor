# TrustAttestor

[English README](./README_EN.md)

TrustAttestor 是一款面向开发者、玩机用户与风控场景的 Android 设备可信度检测工具，用于从系统环境、Root 痕迹、注入行为、Native Bridge、权限漏洞、Key Attestation、吊销列表等多个维度，对设备完整性与运行环境进行综合评估，并以直观的检测结果与评分系统辅助判断设备可信程度。  

---

## 项目简介

TrustAttestor 致力于为 Android 设备提供更直观、更细致的可信度检测能力。项目结合多种本地检测手段与系统环境分析，对设备是否存在异常修改、注入痕迹、可疑权限条件或完整性风险进行综合判断，适合用于安全研究、环境自检、设备状态评估以及风控辅助分析等场景。

## 功能特性

- 多维度设备可信度检测
- Magisk / KernelSu / APatch 等痕迹识别
- Native Bridge / 注入环境检测
- 权限漏洞与系统环境风险分析
- Key Attestation 与吊销列表支持
- 检测评分与可信等级展示

## 适用场景

- 开发者调试与安全测试
- 玩机用户自检设备环境
- 风控与设备可信度辅助判断
- Android 安全研究与实验
